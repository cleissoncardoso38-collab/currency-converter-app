# 💱 Currency Converter App

## 🚀 Sobre o projeto

O Currency Converter App é uma aplicação web desenvolvida para converter valores entre diferentes moedas de forma prática, rápida e intuitiva.

O projeto foi criado com foco em aprimorar habilidades de JavaScript, manipulação de DOM, responsividade e experiência do usuário (UX), além de representar uma evolução de uma versão anterior mais simples do conversor de moedas.

---

## ✨ Funcionalidades

* Conversão de valores entre moedas
* Interface simples e intuitiva
* Atualização visual dinâmica
* Layout responsivo para diferentes dispositivos
* Melhor experiência de navegação para o usuário

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## 🎯 Objetivo

Este projeto foi desenvolvido para fortalecer conhecimentos em desenvolvimento Front-End, especialmente na manipulação de elementos com JavaScript e construção de interfaces funcionais e responsivas.

Além disso, o objetivo foi transformar um projeto de prática em uma apresentação mais profissional para portfólio e futuras oportunidades como Desenvolvedor Full Stack Júnior.

---

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, foram aprimorados conhecimentos em:

* lógica de programação
* manipulação de DOM
* eventos com JavaScript
* responsividade com CSS
* organização de código
* boas práticas de interface visual
